## Descripción del Programa
Este programa fue desarrollado utilizando pseudocódigo en PSeInt.  
Permite introducir las calificaciones de varios estudiantes, calcular el promedio de cuatro notas y determinar si el estudiante aprobó o reprobó la asignatura.

El sistema solicita:
- Cantidad de estudiantes
- Nombre del estudiante
- Cuatro calificaciones

Luego muestra los resultados organizados en forma de tabla indicando:
- Notas
- Promedio
- Estatus (Aprobado o Reprobado)

## Herramienta Utilizada
